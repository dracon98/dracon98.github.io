Gavl - Voice to Bids

Auctions just got lot easier!

New

Our goal was to create an accurate and intuitive application for the purpose of voice to text transcription. For this project, we have worked with GAVL to create an application that takes the vocal cues of an auctioneer and transcribes these to a numerical value that can be confirmed and processed in real time; resulting in a reduction for the need for human accuracy and speed in auction environments. So what makes our application special? Our application is able to transcribe this, in real time, over the cloud and with minimal latency. Meaning in a fast paced bidding environment, everything is captured in real time, reducing human error and making the application far more user friendly and accurate.
